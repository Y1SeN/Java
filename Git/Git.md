# Git

## 一、git工作原理

```text
Workspace：工作区
Index / Stage：暂存区
Repository：仓库区（或本地仓库）
Remote：远程仓库
```

## 二、常用命令

```text
 * 克隆 —— clone
 * 添加 —— add
 * 检出 —— checkout
 * 提交 —— commit
 * 拉取 —— pull
 * 推送 —— push
 * 抓取 —— fetch
 * 分支 —— branch
 * 合并 —— merge
 * 暂存 —— stage
 * 丢弃 —— discard
```
